# 三、表格说明

- 提测质量（感受评分）的打分以提测被驳回次数、需求变更次数、需求插入次数为主要衡量指标

- 提测被驳回次数：未通过冒烟测试的提测次数

- 需求变更次数：需求评审后，测试完成前发生的所有变更次数

- 需求插入次数：需求评审后，测试完成前发生的所有插入次数
