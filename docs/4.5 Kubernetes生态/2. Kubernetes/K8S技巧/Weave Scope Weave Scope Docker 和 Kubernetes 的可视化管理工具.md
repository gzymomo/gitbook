### 软件简介

Weave Scope 用于监控、可视化和管理 Docker 以及 Kubernetes。

Weave Scope 这个项目会自动生成容器之间的关系图，方便理解容器之间的关系，也方便监控容器化和微服务化的应用。

**Weave Scope 的功能特性**

- 拓扑映射
- Scope 中的视图
- 图形或表格模式
- 灵活的过滤
- 强大的搜索
- 实时的应用和容器指标
- 使用插件 API 生成自定义指标
