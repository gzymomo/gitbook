**简要描述：** 

- 商品分类

**请求URL：** 

- `https://api-hmugo-web.itheima.net/api/public/v1/categories`

**请求方式：**

- GET

**参数：** 无

| 参数名   | 类型   |          |
| :------- | :----- | -------- |
| cat_id   | int    | 分类id   |
| cat_name | string | 分类名称 |
| children | array  | 子节点   |
| cat_icon | string | 图标     |
