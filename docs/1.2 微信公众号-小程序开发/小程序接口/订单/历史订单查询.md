**简要描述：** 

- 历史订单查询

**请求URL：** 

- `https://api-hmugo-web.itheima.net/api/public/v1/my/orders/all`

**请求方式：**

- GET

**请求头参数：**

| 参数名        | 必选 | 类型   | 说明                      |
| ------------- | ---- | ------ | ------------------------- |
| Authorization | 是   | string | 用户登录成功获取的token值 |

**请求体参数**：

| 参数名 | 必选 | 类型   | 说明                         |
| ------ | ---- | ------ | ---------------------------- |
| type   | 是   | number | 1:全部订单 2:待付款 3:待发货 |
